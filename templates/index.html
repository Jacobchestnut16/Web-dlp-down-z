Web-dlp-down-z

Web-dlp-down-z is a Flask-based media scraping and management tool powered by yt-dlp. Download individual files or full playlists, view logs, and manage downloads easily—all from your browser. No coding required!

⚙️ Setup Tips

Set your download path in the settings tab.

  • Windows: C:\Users\your-username\Downloads
  • Linux/macOS: ~/Downloads
  • Logs and file lists are auto-managed but configurable.

✍️ Manage Your Downloads

Edit Tab: Organize downloads into groups. Save changes or Save & Execute to start downloading. Install Options:

  • Playlists install to paired download lists.
  • Downloads install to the path you set in settings.
Each entry has a namespace (label for you and logs, this does not name the final file).

✅ Features

  • Download individual videos or entire playlists
  • Auto-sort by source site and uploader
  • Auto-retry failed downloads
  • Format selection (mp3, mp4, webm, etc.)
  • Cross-platform: Windows, Linux, macOS
